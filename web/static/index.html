About

This is the T-220 scaffold console. The static shell is served by the control-plane at /ui/; all data and actions flow through the bearer-gated /v1 API. The token you enter is kept only in this tab's sessionStorage and sent as Authorization: Bearer on each request.

Feature pages (T-221+) extend this by adding a file under web/static/ and a read-model endpoint under internal/host/api/. No build step.
